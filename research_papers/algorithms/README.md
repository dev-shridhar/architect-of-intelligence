# Algorithms Research Papers 🔢

This folder contains seminal papers describing foundational algorithms in deep learning optimization, machine learning models, and system consensus.

## 📄 Papers List

### 1. [Attention Is All You Need (2017)](file:///Users/shridharkulkakrni/.gemini/antigravity-ide/scratch/architect-of-intelligence/research_papers/algorithms/attention_is_all_you_need.pdf)
*   **Authors**: Ashish Vaswani, Noam Shazeer, Niki Parmar, Jakob Uszkoreit, Llion Jones, Aidan N. Gomez, Łukasz Kaiser, Illia Polosukhin.
*   **Impact**: Replaced recurrence (RNNs/LSTMs) with self-attention. This paper is the sole foundation for all modern Large Language Models (GPT, Gemini, Claude, Llama).

### 2. [Adam: A Method for Stochastic Optimization (2014)](file:///Users/shridharkulkakrni/.gemini/antigravity-ide/scratch/architect-of-intelligence/research_papers/algorithms/adam_optimization.pdf)
*   **Authors**: Diederik P. Kingma, Jimmy Ba.
*   **Impact**: Introduced the Adam optimizer, combining momentum and adaptive learning rates to stabilize and speed up training of deep neural networks.

### 3. [In Search of an Understandable Consensus Algorithm (2014)](file:///Users/shridharkulkakrni/.gemini/antigravity-ide/scratch/architect-of-intelligence/research_papers/algorithms/raft_consensus.pdf)
*   **Authors**: Diego Ongaro, John Ousterhout.
*   **Impact**: Introduced the Raft consensus protocol as a cleaner, more understandable alternative to Paxos. Widely used in distributed storage systems like etcd and Consul.
